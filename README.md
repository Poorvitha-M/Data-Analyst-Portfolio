# Data Analyst Internship Portfolio

## Overview

This project demonstrates end-to-end data analysis:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Dashboard Creation
* Data Storytelling

## Tools Used

* Python (Pandas, Matplotlib)
* Excel (Pivot Tables, Charts)
* GitHub

## Key Insights

* Majority customers belong to Consumer segment
* California has highest number of customers
* Average name length is around 13

## Files Included

* Dataset (cleaned_customers.csv)
* EDA Script (eda.py)
* Insights (insights.txt)
* Presentation (PPT)

## Conclusion

This project helped me understand data analysis workflow from raw data to business insights.
